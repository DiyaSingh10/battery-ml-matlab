# Battery Dataset Analysis and ML Regression Modelling

## About
This project performs statistical analysis and machine learning regression 
modelling on a battery dataset using MATLAB and Simulink. The goal was to 
predict battery performance parameters with high accuracy using two ML models.

## Tools & Technologies
- MATLAB
- Simulink
- Artificial Neural Network (ANN)
- Decision Tree Regression

## Methodology
- Performed feature engineering including correlation analysis and 
  distribution metrics to extract key insights from the dataset
- Split data into 80% training and 20% testing sets
- Built and compared two models: ANN and Decision Tree
- Evaluated performance using R², RMSE, and MAE metrics

## Results
- Achieved R² ≈ 0.999 with the optimized Decision Tree model
- Decision Tree outperformed ANN in prediction accuracy on this dataset

## Key Learnings
- Hands-on experience with ML model building and evaluation in MATLAB
- Understanding of feature engineering and statistical data analysis
- Practical knowledge of model comparison and performance metrics
